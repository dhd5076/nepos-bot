# nepos-bot

## Install
Clone the repository
```bash
git clone https://github.com/dhd5076/nepos-bot.git
```
Got to the download location
```bash
cd nepos-bot
```
Install node packages
```bash
npm install
```

## Usage
Got to the install location
```bash
cd nepos-bot
```
Replace config variables
```bash
vim config.js
```
Run the bot
```bash
node index.js
```
